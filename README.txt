
# Bot de Controle Financeiro (Telegram)

## Como usar

1. Crie seu bot no BotFather e copie o token.
2. Renomeie `.env.example` para `.env` e coloque seu token lá.
3. Instale dependências:
   pip install python-telegram-bot python-dotenv
4. Rode com:
   python bot.py

### Comandos disponíveis:
/gastar <valor> <categoria> <descricao> <metodo>
/receber <valor> <descricao>
/start

Mais funcionalidades em breve!
